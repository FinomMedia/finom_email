## Getting Started

```sh
# install the CLI globally
npm i -g @maizzle/cli

# start development
maizzle serve

# build for production
maizzle build production
```

## Working

1. build produkčních emailů 
```sh
maizzle build production
```
2. výsledné HTML soubory ve složce **build_production**
